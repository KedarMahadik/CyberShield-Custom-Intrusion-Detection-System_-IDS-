CyberShield is an automated Python-based Intrusion Detection System (IDS) designed for real-time network security and traffic intelligence. By leveraging Scapy for Deep Packet Inspection (DPI) and Pandas for behavioral data analytics, the system identifies and autonomously mitigates threats such as DDoS attacks and Port Scanning. It bridges the gap between network administration and automation by integrating a proactive threat-response layer that interfaces with host firewalls to block malicious traffic instantly. Designed for enterprise scalability, the project includes a Matplotlib visualization dashboard for rapid Root Cause Analysis (RCA) and security posture auditing.
